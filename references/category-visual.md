# 视觉维度矩阵参考

## 优先级规则

信息来源的优先级（从高到低）：

1. **ICON 品牌色提取**（自动从 ICON_URL 提取）→ 最高，提升为高饱和高明度版本
2. **品牌层级修正**（见下方"品牌层级"章节）→ 命中触发词时在 prompt 末尾追加修正词
3. **分类驱动风格 + 构图**（见 `CATEGORY_RECOMMENDED_STYLES` / `CATEGORY_RECOMMENDED_COMPOSITIONS`）→ 按行号轮选
4. **一级分类基础**（所有分类）→ 基础色调 + 渲染风格 + 构图基调
5. **SUBJECT_POOL 主体池**（兜底）→ 仅当 S_INTRO / HUMAN_DESC 都不可用时使用

冲突处理：ICON 提取的品牌色优先级最高，分类规则仅作补充。

## 过滤规则

- **VISUAL_STOPWORDS**：从 S_INTRO 提取时过滤抽象/功能/品牌词（"软件"、"体验"、"平台"、"系统"、"管理"、"服务"、"腾讯"、"百度"等80+词）
- **LABEL_PATTERNS**：HUMAN_DESC 含以下标签时整句丢弃不可用（"官方"、"安卓"、"手机银行"、"国家税务"、"旗下"、"APP"等）
- **GENERIC_TAGS**：S_INTRO 提取时跳过通用分类词（"常用工具"、"其他工具"、"休闲娱乐"、"数字人"、"安卓"等）
- **字符长度**：S_INTRO 提取词取2-6字中文词组；HUMAN_DESC ≤20字作补充场景，≤24字作主要来源，超出则丢弃

## 内容安全硬性约束（全局注入）

脚本对每张图的 prompt 末尾都会追加以下硬性禁令：

1. **货币**：禁具体国家货币符号 / 币种文字 / 纸币硬币样式 / ￥$€£ → 用中性表达（金色硬币、宝箱、钱袋、礼物盒）
2. **IP**：禁知名 IP / 版权角色（米老鼠、唐老鸭、皮卡丘、超级马里奥、海绵宝宝、小猪佩奇、奥特曼、龙猫、机器猫、漫威/DC 等）
3. **LGBT/合规**：禁彩虹旗、异装造型、性别模糊化表达
4. **暴力/伦理**：禁暴力血腥、色情擦边、歧视性、宗教争议
5. **诱导**：禁误导性医疗效果、金融收益承诺、赌博暗示

非母婴类追加 **视觉成熟度** 约束：严禁 Q 版圆滚滚造型、卡通泡泡字体、幼儿园配色乱炖、幼稚化图形、可爱化大眼萌。

## ⚠️ 游戏类全局风格禁用（重点强调）

**作用域**：`PARENT_ID == "游戏"` 的全部应用（含 11 个子分类：动作冒险 / 飞行射击 / 角色扮演 / 经营策略 / 休闲益智 / 棋牌游戏 / 体育竞速 / 其他游戏 / 特色分类 / 音乐舞蹈 / 打飞机）。

**禁用清单**：

| 风格 | 名称 | 禁用理由 |
|---|---|---|
| **#7** | 3D 极简玻璃拟态 | 玻璃/透明质感无法承载力量感/战斗感；该风格本身严禁任何人物/肢体 |
| **#8** | 新拟物 2.0 充气 3D | 充气可爱感与游戏对抗/史诗调性彻底冲突；同样禁人物 |
| **#12** | Corporate Memphis 扁平插画 | 糖果色卡通营销风，与游戏视觉张力不兼容 |

**三重保险机制**（脚本层）：

1. `CATEGORY_RECOMMENDED_STYLES` 内所有游戏子分类的推荐池**预先剔除** 7/8/12
2. `pick_style_for_category` 函数内**二次过滤** `[s for s in recs if s not in GAME_BANNED_STYLES]`
3. `force_style`（手动指定）+ 全局轮换兜底路径在 `main()` 中通过 `while` 循环**强制顺延**

即使用 `--style 7` 强制指定，游戏类应用也会自动跳到风格 9 或 10。
