# 构图矩阵参考

## 使用规则

构图分配优先级：
1. **风格 12** → 5 种 Memphis 居中构图（随机）
2. **分类驱动** → 命中 `CATEGORY_RECOMMENDED_COMPOSITIONS` 时，按 `行号 mod 构图池大小` 轮选
3. **随机兜底** → 未登记分类从下面 10 种通用构图随机取一种

手动覆盖：用户附加 `构图=宏大仰视式` 等指令时，直接使用指定方案。

---

## 10 种构图方案

| 构图名 | 视角描述 | 主体占比 |
|--------|---------|---------|
| 宏大仰视式 | 极低角度仰视，主体渺小位于画面中央，天空占画面60-70%云雾层次 | 30-40% |
| 低角冲击式 | 低角度俯视，地面纹理向远处延伸，主体位于画面中央，前景冲击力强 | 30-40% |
| 中心爆发式 | 中心点放射构图，能量向外扩散，主体位于画面中央，四周能量波纹粒子离心 | 30-40% |
| 叙事留白式 | 主体位于画面中央，背景左右不对称留白，环境元素偏向一侧营造叙事感，前简后繁 | 45-60% |
| 对角线动态式 | 主体位于画面中央，背景对角线光轨元素流向主体，运动感由背景元素承载 | 45-60% |
| 中心稳重式 | 主体位于画面中央微偏上，对称平衡，四周环绕弱散射光 | 45-60% |
| S曲线探索式 | 主体位于画面中央，背景S形光流路径环绕主体延伸，前景清晰中景过渡背景梦幻 | 45-60% |
| 特写冲击式 | 极近距离特写，主体充满画面居中，背景完全虚化仅保留色彩氛围 | 70-80% |
| 英雄正面式 | 正面平视，主体居中充满画面中央，背景简洁渐变主体边缘发光 | 70-80% |
| 仰角特写式 | 低角度仰视特写，主体压迫感强位于画面中央，背景天空或光晕主体底部延伸出画面 | 70-80% |

---

## 主体定位全局约束

- 主体视觉锚定画面**垂直中心或偏上三分之一**位置，严禁主体下沉至画面下半部造成顶部大面积空白
- 主体重心保持在画面纵向 **35%-60%** 区间
- 主体需延伸占满画面高度的 **60% 以上**
- 顶部区域必须有背景层次元素（极度模糊的场景物件/光晕/流光装饰）填充，不得呈现单调纯色大片空白
- 画面饱满度 **≥75%**，严禁任何区域（顶部/左右/角落）出现大面积空旷留白

## 差异化构图推荐（按分类）

| 分类 | 推荐构图 | 人物形式 |
|------|---------|---------|
| 游戏-动作冒险 | 仰角特写式 / 英雄正面式 | 背影/侧身剪影 |
| 游戏-休闲益智 | 低角冲击式 / 中心爆发式 | 可添加游戏角色背影 |
| 游戏-经营策略 | 宏大仰视式 / 中心爆发式 | 不添加 |
| 游戏-棋牌 | 中心稳重式 / 英雄正面式 | 不添加 |
| 应用-考试.学习 | 叙事留白式 / 中心稳重式 | 不添加，知识道具为主 |
| 应用-视频软件 | 对角线动态式 / S曲线探索式 | 拍摄的手（局部） |
| 应用-社交 | 叙事留白式 / 中心稳重式 | 背影/手部 |
| 应用-工具 | 叙事留白式 / 中心稳重式 | 操作者局部 |
| 应用-购物 | 对角线动态式 / 英雄正面式 | 购物的手（局部） |
| 应用-生活.服务 | 宏大仰视式 / 叙事留白式 | 可添加服务场景背影 |
| 应用-母婴.儿童 | S曲线探索式 / 宏大仰视式 | 儿童卡通人物（正面可用） |

## 差异化生成规则

1. 分类驱动构图按 `行号 mod 构图池大小` 轮选，同分类相邻应用自动分到不同构图
2. 同一应用多次生成时，第二次必须更换构图方案
3. 构图根据画面内容适配，矩阵仅为默认值，可覆盖
4. 人物不强制：场景合理则添加，不合理则不添加
