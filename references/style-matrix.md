# 风格17详细规范：2D扁平插画 + 数字颗粒质感

通过高饱和几何图形、动态图形构成与矢量化表现，营造具有未来运动科技感的品牌级2D扁平插画视觉。少而精，图形感优先，动态优先，品牌统一性优先，视觉节奏优先。

适用场景：运动活动KV、科技品牌海报、APP启动页、运营插画、AI能力展示、潮流活动视觉、智能推荐Banner

核心风格标签：2D Flat Illustration / Geometric Futurism / Vector Graphic Design / Contemporary Editorial Illustration / Sports Tech Visual / Graphic Motion Aesthetic

视觉情绪：年轻化 / 未来感 / 运动感 / 科技感 / 活力 / 高能量 / 数字化 / 国际化品牌视觉 / 简洁动态 / 未来运动感

---

## 色彩系统

- 主色（1~2种，覆盖60%）：荧光绿 / 电子蓝 / 明亮黄 / 青绿色 / 珊瑚橙 / 桃粉色 / 深海军蓝（按图标主色优先选取）
- 辅色（1种，覆盖30%）：与主色形成高饱和冷暖撞色，纯色大面积铺色
- 强调色（1种，覆盖10%）：数字感荧光色，仅用于视觉焦点局部强调
- 全画面主要颜色不超过5种；高饱和色面积不超过画面70%
- 禁止彩虹式平均配色；禁止多种高纯度色同时大面积出现
- 推荐配色方案：蓝+青+荧光绿 / 黄+绿+深蓝 / 橙+蓝+白 / 青蓝+桃粉+深紫

## 颜色活力校正

图标主色往往偏灰偏暗（App图标设计追求克制），直接用于海报会让画面沉闷，因此提取后需要映射到高饱和版本：

| 原色系 | 映射目标 |
|--------|---------|
| 绿/黄绿系 | 荧光绿 `#00e060` / 嫩绿 `#80e040` |
| 橙/棕系 | 活力橙 `#ff6020` / 亮橙 `#ff8c00` |
| 黄/土黄系 | 柠檬黄 `#d4e000` / 明黄 `#ffe040` |
| 红/暗红系 | 鲜红 `#ff2040` / 珊瑚红 `#ff5040` |
| 蓝/灰蓝系 | 电子蓝 `#0080ff` / 青蓝 `#00c0e0` |

确保主色：高明度（亮度>50%）+ 高饱和（饱和度>70%），禁止灰棕色、泥土色、褪色系、暗沉色直接作为主色。

## 线条与形状

- 线条：极简矢量线条，边缘干净有力，几何化轮廓
- 形状语言：**圆角为主（70%），锐角为辅（30%）**——主体与背景色块带适度圆角营造亲和感，速度线/放射线保持锐利几何感提供速度与力量
- 圆角半径控制在形状宽度的 20~30%，有圆润感但不变成气球/泡泡感
- 整体定调：**Rounded Geometric Futurism**，圆润亲和为主基调，锐角速度线作点缀
- 禁止：全画面所有元素均为锐角 / 主体变形成气球泡泡 / 圆角消失变成纯硬朗几何

## 人物剪影规则

风格17中的人物必须是高度简化的抽象几何剪影，禁止写实人物或CG人物。原因：2D扁平风格的核心是图形感和品牌统一性，写实人物会破坏简洁克制的视觉语言。

- 抽象人物剪影最多使用 2~3 个色块，不得出现细碎分割
- 色块划分：头部1色（圆形/椭圆）+ 身体1色（矩形）+ 可选四肢轮廓1色
- 每个色块面积要大且清晰，颜色使用画面辅色或点缀色
- 人物必须与主体产生明确动词互动（冲刺/飞跃/推进/骑乘/挥手/奔跑）
- 人物比例：高度不超过画面高度的40%，作为动态辅助元素而非主体
- 若人物与主体无法形成自然和谐的叙事关联，必须省略人物
- 禁止：服装纹路细节 / 面部五官精描 / 超过3个色块 / 内部多色分割

## 主体占比与构图

- 主体占比：50~60%，位置居中偏上
- 主构图方式：低机位超广角仰视 + 斜向动态构图，主体具有明确前冲/向上趋势
- 速度线密集放射，粗细不一形成层次感；流线飘带从主体后方大幅延伸
- 放射线从主体中心向四周扩散
- 全画面只有1个主视觉中心，边缘细节低于主体
- 禁止：完全对称构图 / 主体静止站立 / 元素水平平均排布 / 速度线过少缺乏动感

## 背景规则

- 背景最多3~4组大面积纯色块，留白比例不低于20%
- 碎片化元素数量≤3个，不得出现4个以上独立小装饰图形散落
- 几何图形之间只允许覆盖（前层叠加后层），禁止相互切割产生交叉碎片
- 全画面独立色块总数不超过8个主要色块（含主体+背景）
- 明亮浅色调（浅蓝/浅灰渐变），避免纯白/纯黑

## 光线与纹理

- 光线类型：图形化光影（Graphic Lighting），非真实物理打光
- 特征：硬边阴影 / 矢量渐变光感 / 局部边缘发光
- 纹理：轻颗粒质感 / 数字噪点 / 半调网点（低密度，不堆叠）
- 禁止：PBR材质 / 真实世界光照逻辑 / 物理打光

## 必须包含

2D扁平插画 / 矢量感 / 高饱和色彩 / 几何图形构成 / 动态透视 / 干净边缘 / 纯色块设计 / 图形化光影 / 轻颗粒纹理 / 现代品牌视觉 / 统一运动方向 / 单一视觉中心

## 必须禁止

- 内容：文字/数字/字母/Logo/水印、真人写实人物、手机等设备、画中画/相框
- 虚拟主体黑名单（禁止作为主体物）：光束 / 光幕 / 光效 / 光带 / 光环 / 粒子 / 能量
- 构图：完全对称构图 / 元素铺满画面 / 所有区域高细节 / 多视觉中心竞争
- 色彩：超过4种主要颜色 / 彩虹式平均配色 / 大量荧光色同时满铺
- 风格：写实风格 / 3D渲染感 / 电影级光影 / 厚涂 / 赛博朋克霓虹 / 低饱和灰暗 / PBR材质 / 气球泡泡感

# 风格18详细规范：企业孟菲斯风格 v7

企业孟菲斯2D扁平矢量插画。超高亮度荧光纯色背景 × 粗黑等粗轮廓线 × 大量元素密集叠压交织 × 小彩色几何体散落，构建视觉饱满、层次丰富的高凝聚力品牌图腾。

## 二、元素密度与叠压法则

**元素数量下限：**
- 主体物：1个（大，居中）
- 人物：1-3个（与主体深度交织）
- 配件/道具：3-5个（穿插在人物和主体之间）
- 小几何装饰体：6-10个（散落在主体团外围）
- 合计：最少15个视觉元素，形成密集叠压的饱满视觉团

**叠压深度要求：**
- 所有主要元素必须相互叠压，前景遮挡后景
- 配件/道具从主体和人物的夹缝中穿插而出
- 人物四肢必须穿插在主体物周围，不是孤立站在旁边

**小几何装饰体规范：**
- 形状：小方块、圆形、菱形星、三角等混合
- 尺寸：大中小三档混合（主体10%、6%、3%各若干）
- 角度：各装饰倾斜不同角度（30°/45°/60°/15°）
- 位置：主体团外围轻盈散落，不贴边不出框

张力来源于叠压密度，禁止使用速度线/放射线/光效。

## 三、构图类型（5种，批量生成时循环）

| id | 构图名称 | 创意核心 |
|----|---------|---------|
| 0 | 极度特写式 | 人物局部超大特写（手/臂）占65%，手持/托举/指向主体，大量元素在手周围叠压散落 |
| 1 | 人物融合式 | 人物与主体深度交织不可分割，四肢穿插环绕，配件从夹缝穿出 |
| 2 | 浮空聚拢式 | 主体浮空居中，人物从下方托举，8-10个彩色几何装饰轻盈散落 |
| 3 | 极角对角线式 | 主体在右上，人物在左下，强对角张力，配件和装饰沿对角线密集排布 |
| 4 | 多人包围式 | 2-3个人物从不同方向环绕主体，各持不同道具，形成丰富叙事 |

共同基础：低机位超广角，图形化透视，所有元素向中心聚拢，四周留白。

## 四、线条与阴影

- 所有主要元素（主体/人物/配件）：1.5-2.5pt 等粗黑色闭合轮廓线
- 小几何装饰体：1-1.5pt 黑色轮廓线
- 阴影：平面切分阴影，深一阶纯色块，硬边无羽化
- 光线：图形化平面光
- 纹理：背景绝对平滑（可加极细微数字噪点）
- 禁止：速度线/光效/发光/模糊/羽化/无轮廓/PBR材质/真实物理打光

---

## 必须包含与禁止

**必须包含（✅）：**
- 超高亮度荧光纯色背景（明度>70%，饱和度>90%）
- 全画一个强调色贯穿主体细节+服装+部分装饰
- 最少15个叠压元素构成密集饱满视觉团
- 小彩色几何装饰体（方块/圆/菱形/星，大中小混合，不同角度）
- 人物与主体深度交织（四肢穿插/托举/环绕）
- 粗黑等粗轮廓线1.5-2.5pt覆盖所有主要元素

**严苛禁令（❌）：**
- 禁止速度线/放射线/光效/光束
- 禁止背景低亮度
- 禁止元素稀少（不能只有3-5个元素）
- 禁止人物孤立站在主体旁
