# 风格矩阵参考

## 使用规则

风格分配优先级：
1. `--style N` 强制指定（命中游戏禁用自动顺延）
2. **分类驱动**：按 `CATEGORY_RECOMMENDED_STYLES`（见 category-visual.md）查找该分类的风格池，按 `行号 mod 池大小` 轮选
3. **全局轮换兜底**：未登记分类按 `1→2→3→5→6→7→8→9→10→11→12→1` 顺序（始终跳过 4；游戏类额外跳过 7/8/12）

特殊处理：
- **风格 4 羊毛毡**：母婴类主推（在母婴风格池 `[4, 9, 3, 2]` 中按行号轮选）；其他分类不会用到
- **风格 7/8/12**：⚠️ **游戏类全局禁用** — `PARENT_ID == "游戏"` 的全部 11 个子分类（动作冒险/飞行射击/角色扮演/经营策略/休闲益智/棋牌游戏/体育竞速/其他游戏/特色分类/音乐舞蹈/打飞机）一律禁用；强制指定 / 全局轮换 / 分类推荐三条路径均自动顺延
- **风格 11/12**：**高端品牌不宜**（卡通糖果色与高端调性冲突），需手动选择时建议同时把品牌层级降为标准

## 补色/点缀色规则（风格1-11通用）

每张图随机从以下4种色彩关系中选一种，**严格控制面积在画面5%以内**：

| 模式 | 色相差 | 示例 |
|------|--------|------|
| 互补色 | 150-180° | 红→青绿，蓝→橙，黄→紫 |
| 对比色 | 120-150° | 红→蓝绿，蓝→黄橙，橙→蓝紫 |
| 撞色 | 90-120° | 红→紫，蓝→黄绿，橙→蓝 |
| 邻近对比色 | 60-90° | 红→黄，蓝→绿，橙→绿 |

**融入方式**（严禁独立漂浮几何体）：
- ① 主体边缘轮廓逆光（rim light）偏向点缀色色温
- ② 发光屏幕/内发光散射光晕带点缀色温
- ③ 材质折射/玻璃色散/半透明透射产生点缀色光斑
- ④ 冷暖分色中阴影面渗入点缀色调

## 风格1&2 背景处理专项规范

背景层次由**两类元素**共同营造，两者色差均控制在背景色10%以内：

1. **超大面积同色系柔光漫射**：光区占背景50%以上，光与背景之间无任何可见边界（极超大范围羽化完全融化消失如浓雾散射），**严禁**出现任何光边轮廓线或可辨识的光束形状
2. **极大景深完全模糊的背景元素**：根据画面场景内容自然选择匹配物体形态，模糊至仅剩色彩轮廓无任何细节，营造纵深感

**人物风格（风格1&2成人场景）**：成熟商业插画风格，成人正常头身比例（1:6至1:7），五官精致立体但不夸张，整体呈现高端商业插画感；严禁迪士尼/皮克斯卡通大眼萌感造型、头部比例夸张、儿童化圆滚滚脸型、真实照片感人脸毛孔皮肤纹理

## 风格12 现代扁平插画专项规范

### 配色系统（严格4色）
- ① 背景主色（≥65%铺满全画）：品牌色高饱和鲜亮纯色（S 82-92%，L 45-65%），完全平涂干净背景，无任何纹理渐变
- ② 主对比色（约20%）：选1个与品牌色强对比的鲜亮色（色相差90-150°，S 80-90%，L 42-62%），用于主体物主色和关键图形
- ③ 高亮白/近白（约10%）：白色或极高明度色（L 92-98%），用于主体物受光面和高光细节
- ④ 主色深版阴影（约5%）：与主色相同色相但明度降低（L 18-32%），专用于装饰元素阴影

严禁超过4种主色（含白不含黑），严禁灰调/莫兰迪/大地色/任何渐变，所有色块纯色平涂

### 主体造型规范
- 白色或浅色大圆角造型，粗黑 1.5-2.5pt 等粗闭合轮廓线，无内部细节堆砌，平面切分硬边阴影（无羽化）

### 构图规范（5种循环，全部严格水平居中）

**核心约束（所有构图通用）**：
- 所有主体元素必须聚合为一个整体视觉团，**水平方向严格居中（中心线±10%以内）**
- 若有2个主体，两者必须物理接触或重叠，间距为零，形成不可分割的单一视觉团
- **严禁将2个主体分放在画面左右两侧/上下两端**

| 构图名 | 说明 |
|--------|------|
| 主体居中饱满式 | 整体团块位于画面绝对正中央，占画面55-70%，2主体紧贴合并 |
| 主体居中仰角式 | 整体从中央偏下向上延伸，仰角冲击感，水平严格居中，占高度60-70% |
| 主体居中偏上式 | 整体位于水平中央+垂直偏上1/3，2主体紧密叠放，占画面55-65% |
| 主体居中前倾式 | 整体略向前倾10-15°，位于正中央，2主体紧贴无间隙，占画面60-70% |
| 主体居中充满式 | 充满画面中央区域，水平垂直均居中，2主体重叠，占画面65-75% |

### 氛围规范（3种循环）— Corporate Memphis style22
- **放射爆发型**：以主体物为原点向四周发射几何线条、星形和气泡，从主体后方延伸出放射状色块（Ray lines）和发散的小几何粒子，与速度线协同营造能量爆发感，使用背景同色系深一阶
- **磁场轨迹型**：背景装饰元素表现为平滑等粗曲线（Path lines），沿椭圆轨道环绕主体向外扩散，营造连接与流动感，使用背景同色系深一阶
- **几何底座型**：画面底部有同色系深一阶的厚实几何块面或重叠半圆作为几何底座，承托主体根部，高度约10-15%，营造稳重生长感

### 装饰元素规范
- 3-5个与应用内容相关的几何图标
- 使用背景同色系深浅变体细线描边
- 沿速度线/放射线方向向外发散，严禁密集堆砌或漂浮四角

### 其他约束
- 描边：主体物及所有元素统一 1.5-2.5pt 等粗黑色闭合轮廓线，全画一致
- 背景绝对干净，无任何纹理/图案/渐变覆盖
- **人物（有人物时）**：Corporate Memphis风格，小头长肢巨型末端，点状眼极简弧线嘴，肤色米色/浅棕/橙棕（严禁绿/蓝/紫/灰），1.5-2.5pt黑色轮廓线，必须与主体物物理接触/重叠并有明确互动动作，约占内容区20%；无法与主体自然关联则省略
