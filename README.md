# App Store Cover Generator

输入 App 信息，自动生成应用商店封面展示图。

## 它能做什么

只需提供 App 名称、分类和一句话介绍，自动完成：

- **信息补全** — 信息不足时自主推断视觉方向，不追问
- **风格匹配** — 内置 18 种视觉风格，每次出图自动轮换确保多样性
- **主体推导** — 从核心功能提取差异化视觉符号
- **质量自检** — 自动评分，不达标自动修正
- **生图输出** — 调用配置的 API 直接出图

## 效果示例

```
用户输入：
  美团 · 生活服务 · 外卖美食买菜打车酒店火车票

自动处理：
  品牌层级 → 标准
  视觉风格 → 好莱坞科幻大片超级写实风格 (金蓝辉光/星云紫)
  构图方案 → 天空主导式 (天空60% + 黄金时刻逆光)
  主体推导 → 装满美食的外卖箱，食材欢快飞出
  叙事节点 → 结果时刻 — 一家人围坐享受美食

质量自检：
  主体清晰度 4/5  光线描述 4/5  构图完整性 4/5
  风格一致性 4/5  品牌色落地 4/5  可生成性 4/5
  总分 40/50 ✅ 通过

输出：
  → 16:9 横图，可直接用于应用商店首页
```

## 能处理多模糊的输入

| 输入信息 | 处理方式 |
|---------|---------|
| 名称 + 分类 + 简介 | 直接生成 |
| 名称 + 分类 | 语义推断补全 |
| 只有名称或只有分类 | 双引擎推断 |
| 一句模糊描述 | 生成 3 个风格方向供选择 |

## 配置

创建 `.env`，填入生图 API：

```env
IMAGE_API=<openai / replicate / fal>
IMAGE_BASE_URL=<你的 API 地址>
IMAGE_API_KEY=<你的 Key>
IMAGE_MODEL=<模型名>
```

支持 OpenAI 兼容接口及 Replicate、Fal.ai。未配置时自动引导配置。
