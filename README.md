# App Store Cover Generator

输入 App 信息（CSV 批量），自动生成 16:9 横版应用商店封面图。

---

## 🚀 快速开始（5 分钟跑通第一张图）

### 1. 准备环境
```bash
# Python 3.9+
python3 --version

# 克隆/解压本项目后，进入项目目录
cd app-store-cover-generator

# 安装依赖（建议先创建虚拟环境）
python3 -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

### 2. 配置 API Key（二选一）
```bash
# 方式 A：环境变量（推荐，不留文件）
export PACKY_API_KEY='sk-your_real_key_here'

# 方式 B：.env 文件（适合本地长期开发）
cp .env.example .env
# 然后编辑 .env，填入 PACKY_API_KEY
```

> 申请 Packy API Key：<https://www.packyapi.com>（兼容 OpenAI 协议，默认模型 `gpt-image-2`）

### 3. 准备 CSV 数据
CSV 必须包含以下列（中文括号写法也支持）：
```
PACKAGE_NAME, APP_NAME, S_INTRO, ICON_URL, NAME, PARENT_ID
[或]   包名(PACKAGE_NAME), 应用名称(APP_NAME), 简介(S_INTRO), 图标URL(ICON_URL), 分类(NAME), 父分类ID(PARENT_ID)
```

将文件放到 `data/input.csv` 或运行时用 `--csv` 指定路径。

### 4. 跑第一批
```bash
# 跑 CSV 第 1-5 行
python3 scripts/batch_generate.py --start 1 --end 5

# 指定 CSV 和输出目录
python3 scripts/batch_generate.py \
  --csv /path/to/your.csv \
  --outdir /path/to/output \
  --start 1 --end 20

# 强制指定风格（1-12，游戏类禁用 7/8/12 自动顺延）
python3 scripts/batch_generate.py --start 1 --end 5 --style 9
```

输出默认存到 `output/covers/`，文件名格式 `<PACKAGE_NAME>.jpg`，单张 ≤ 200KB。

## 它能做什么

只需提供 App 名称、分类和一句话介绍，自动完成：

- **品牌色提取** — 从 ICON 提取主色，提升为高饱和高明度版本
- **主体推导** — S_INTRO 提取2词 → HUMAN_DESC 补充 → SUBJECT_POOL 兜底
- **分类驱动风格** — 每个分类有专属风格池，按行号轮选（如母婴用 [4羊毛毡, 9粘土, 3CG厚涂, 2 3D立体]）
- **分类驱动构图** — 每个分类有推荐构图组合（如视频软件用对角线动态/S曲线探索）
- **品牌层级修正** — 自动识别高端（大厂/央媒）/精品（国际授权）品牌，注入对应的克制电影感/精品插画感
- **内容安全约束** — 全局禁止具体货币符号、知名 IP、LGBT 元素、医疗金融误导；非母婴类强制视觉成熟度
- **生图输出** — 调用配置的 API 直接出图，压缩至 ≤200KB

## 各风格核心规则

### 风格 1 · 写实3DCG画风格
- 哑光珐琅质感（S 45-62%，L 50-65%），表面哑光无镜面反射
- **背景双层结构**：①超大面积同色系柔光漫射（无可见边界，浓雾散射），②极大景深完全模糊的背景元素（仅剩色彩轮廓）
- **补色活力**：主体背光面/暗部边缘施加互补色调（色相差≥120°）柔和环境反射光；屏幕/发光部件带补色温内发光
- 严禁：独立漂浮几何体呈现补色、白色光源、硬阴影、镜面高光

### 风格 2 · 3D立体风格
- 磨砂金属质感（S 30-48%，L 45-60%），微粒漫反射
- 背景同风格1双层结构，色差≤8%
- **补色活力**：主体背面/侧面补色调柔和逆光边缘光（rim light），磨砂面隐含补色光感
- 严禁：白色泛光、多光源叠加、独立漂浮几何元素

### 风格 3 · CG厚涂风格
- S 58-76%，L 52-68%，色彩鲜明不发灰
- **冷暖分色**：受光面主色暖调高光，背光面/阴影边缘渗入互补色冷调（色相差≥120°）
- 强侧硬光、笔触光影、硬边阴影

### 风格 4 · 羊毛毡风格 ⚠️ 仅母婴/儿童
- 高明度柔和色系（S 30-55%，L 70-85%）
- 触发关键词：母婴/儿童/婴幼/幼儿/宝宝/亲子/启蒙

### 风格 5 · 壁画质感风格
- 复古艺术色调（S 52-68%，L 52-65%），饱满有层次，禁止发灰发黑
- **复古光感活力**：天窗丁达尔散射光带补色冷暖偏移，光束本体主色暖调、边缘渗入补色冷调；材质表面交界处出现彩虹薄膜反射光
- 光束边缘必须大范围柔化扩散，严禁清晰硬边光束轮廓线

### 风格 6 · 轻量超现实3D商业插画
- 主色中高饱和（S 50-65%），背景极高明度（L 80-92%），清新通透
- **背景充实**：叠放 1-2 个超大体量相关元素，极深度高斯模糊（仅剩极淡轮廓），色差≤8%
- **补色活力**：主体边缘补色调柔和侧面环境光，屏幕内发光带补色温
- 正面+顶部品牌色有色柔和漫射光，禁止纯白光源

### 风格 7 · 3D极简玻璃拟态
- S 48-68%，L 58-75%，背景同色系极简双色渐变（L≥55%）
- **核心**：主体物直接置于背景上，**严禁任何容器/面板/气泡/玻璃球包裹**；玻璃质感仅体现在主体物自身材质（折射/透明/高光边缘）
- **折射活力**：主体物边缘色散出补色彩虹薄边，焦散光斑散落周围
- 焦散效果、折射光、软阴影、全局照明GI
- ⚠️ **严禁出现任何人物/人形/人体/人脸/手/肢体**，主视觉必须为产品/道具/物件/场景元素

### 风格 8 · 新拟物主义2.0充气3D
- 柔和梦幻渐变（S 52-68%，L 58-72%），背景明亮版（L≥52%），禁止荧光色
- **充气透光活力**：充气体半透明材质，背光侧透出互补色透射光（S 45-60%），边缘呈现冷暖光晕过渡
- 丁达尔效应大范围弥散，无硬边光束
- 严禁：密集缠绕曲线/盘旋流线/复杂螺旋装饰
- ⚠️ **严禁出现任何人物/人形/人体/人脸/手/肢体**，主视觉必须为产品/道具/物件/场景元素

### 风格 9 · 3D粘土C4D潮流玩具
- 主色同色系渐变（S 58-72%，L 55-68%），背景均匀单色（L≥52%）
- **背景**：叠放 1-2 个超大体量物件高斯模糊（色差≤12%），**严禁背景径向渐变光晕/圆形聚光/中心亮斑**
- **粘土光感活力**：粘土主体侧光面/背光面边缘补色轮廓逆光（rim light）
- 单侧45°大柔光漫射，光区均匀扩散

### 风格 10 · 潮流3D艺术混合材质拟物
- S 50-68%，L 55-70%，背景深饱和版（L≥42%，S≥55%），禁止背景发黑发灰
- **混合材质活力**：主体内部自发光本体主色系，外散射时色温偏移向补色，外轮廓/透明材质边缘形成补色发光晕圈；金属玻璃交界面折射产生补色色散
- 柔和全局光+强边缘逆光+内部自发光

### 风格 11 · 2D扁平插画 Rounded Geometric Futurism
- 品牌色约60%（S 52-72%，L 55-72%）+ 辅色20% + 点缀色10%，**全画不超过4色**
- 数字颗粒+半调网点质感，硬边色块阴影
- 主体轮廓局部边缘发光

### 风格 12 · 现代扁平插画风格（严格4色系统）
- **配色**：
  - ① 背景主色（≥65%）：品牌色高饱和纯色（S 82-92%，L 45-65%）平涂
  - ② 主对比色（约20%）：色相差90-150°，S 80-90%，L 42-62%
  - ③ 高亮白/近白（约10%）：L 92-98%
  - ④ 主色深版阴影（约5%）：同色相 L 18-32%，专用于装饰元素阴影
- **主体定位（最高优先级）**：所有主体聚合为一个整体视觉团，**水平方向严格居中（中心线±10%以内）**，整体占55-70%；若有2个主体，必须物理接触/重叠，**严禁分放左右两侧/上下两端**
- **专属构图**：5种居中构图随机取用（主体居中饱满式/仰角式/偏上式/前倾式/充满式），所有构图均强制水平居中
- **主体造型**：白色或浅色大圆角造型，粗黑 1.5-2.5pt 等粗闭合轮廓线，无内部细节堆砌，平面切分硬边阴影（无羽化）
- **描边**：主体物及所有元素统一 1.5-2.5pt 等粗黑色闭合轮廓线，全画一致
- **装饰元素**：3-5个与应用内容相关的几何图标，背景同色系深浅变体细线描边，沿速度线/放射线方向向外发散
- **氛围**（3种随机）：放射爆发型 / 磁场轨迹型 / 几何底座型（均使用背景同色系深一阶）
- **人物（有时）**：Corporate Memphis风格，小头长肢巨型末端，点状眼极简弧线嘴，肤色米色/浅棕/橙棕，必须与主体物物理接触/重叠并有互动动作；无法自然关联则省略

## 内容安全硬性约束（全局注入）

每张图的 prompt 末尾都会追加以下硬性禁令（违反作废）：

- **货币**：禁具体国家货币符号 / 币种文字 / 纸币硬币样式 / ￥$€£符号 → 用中性表达（金色硬币、宝箱、钱袋）
- **IP**：禁知名 IP 形象 / 版权角色（米老鼠、皮卡丘、超级马里奥、海绵宝宝、小猪佩奇、奥特曼、龙猫等）
- **LGBT**：禁彩虹旗符号 / 异装造型 / 性别模糊化表达
- **暴力/合规**：禁暴力血腥 / 色情擦边 / 歧视性 / 宗教争议
- **金融医疗**：禁误导性医疗效果、金融收益承诺、赌博暗示

**视觉成熟度**（非母婴类追加）：严禁 Q 版圆滚滚造型、卡通泡泡字体、幼儿园配色乱炖、幼稚化图形、可爱化大眼萌。

## 全局约束（所有风格通用）

| 类别 | 规则 |

## 主体推导优先级

```
① S_INTRO → 提取2个具体可视化词 + HUMAN_DESC（≤20字）补场景氛围
② S_INTRO 只提取到1个词 → 结合 HUMAN_DESC（≤20字）补场景
③ S_INTRO 无有效词 → 根据 NAME 分类提取 HUMAN_DESC（≤24字）+ App名产品场景
④ SUBJECT_POOL 分类主体池兜底（19个分类专属主体列表）
⑤ 通用兜底：{App名}核心功能视觉符号
```

**过滤规则**：
- **VISUAL_STOPWORDS**：过滤抽象/功能描述词（"软件"、"体验"、"平台"等80+词）
- **LABEL_PATTERNS**：HUMAN_DESC 含机构标签（"安卓"、"官方"、"国家税务"等）时整句丢弃
- **GENERIC_TAGS**：通用分类词从 S_INTRO 提取时跳过（"常用工具"、"其他工具"、"休闲娱乐"等）
- **字符长度过滤**：S_INTRO 提取词仅取2-6字中文词组；HUMAN_DESC ≤20字才作为补充场景氛围，≤24字才作为主要来源

## 点缀色色彩关系（每张随机取一种）

**面积约束**：点缀色总面积严格控制在画面的 **5% 以内**，仅作为光感层次点缀，禁止大面积铺开。

**融入方式**（只能通过光的物理方式，严禁独立漂浮几何体）：
- ① 主体边缘轮廓逆光（rim light）偏向点缀色色温
- ② 发光屏幕/内发光的散射光晕带点缀色温
- ③ 材质折射/玻璃色散/半透明透射产生点缀色光斑
- ④ 冷暖分色中阴影面渗入点缀色调

| 模式 | 色相差 | 示例 |
|------|--------|------|
| 互补色 | 150-180° | 红→青绿，蓝→橙，黄→紫 |
| 对比色 | 120-150° | 红→蓝绿，蓝→黄橙，橙→蓝紫 |
| 撞色 | 90-120° | 红→紫，蓝→黄绿，橙→蓝 |
| 邻近对比色 | 60-90° | 红→黄，蓝→绿，橙→绿 |

## 运行

```bash
# 默认（CSV_PATH 内置 + 前20条 + 默认输出目录）
python3 scripts/batch_generate.py

# 指定 CSV、范围、输出目录
python3 scripts/batch_generate.py \
  --csv "/path/to/data.csv" \
  --start 1 --end 52 \
  --outdir ~/Desktop/your_folder

# 强制使用某个风格（自动跳过游戏类禁用风格）
python3 scripts/batch_generate.py --style 9
```

**命令行参数：**

| 参数 | 默认 | 说明 |
|------|------|------|
| `--csv` | 脚本内 `CSV_PATH` | CSV 文件路径 |
| `--start` | 1 | CSV 起始行号（1-based，含） |
| `--end` | 20 | CSV 结束行号（1-based，含） |
| `--style` | 0 | 强制风格 1-12（0=自动轮换） |
| `--outdir` | 脚本内 `OUT_DIR` | 输出目录 |

输出文件名格式：`PACKAGE_NAME.jpg`
