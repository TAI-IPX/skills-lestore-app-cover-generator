# App Store Cover Generator

面向设计师和运营，解决应用商店封面图批量出图的效率问题。只需提供 App 名称、分类和一句话简介，工具会自动提取品牌色、匹配视觉风格、生成符合平台规范的封面图，单张压缩至 200KB 以内。

![效果预览](preview.png)


## 功能

- 🎨 **品牌色提取** — 从 ICON 提取主色，自动提升为高饱和高明度版本
- 🖼 **分类驱动风格** — 12 种视觉风格，按 App 分类自动匹配，支持强制指定
- 🧠 **主体智能推导** — 从简介提取可视化词，多级兜底确保主体不为空
- 🏷 **品牌层级识别** — 自动识别大厂/精品品牌，注入对应视觉调性
- 🔒 **内容安全约束** — 全局过滤货币符号、版权 IP、违规元素
- ⚡ **批量生成** — 从数据源批量读取，按行号轮选风格，支持断点续跑


## 工作原理

```
App 信息（名称 + 分类 + 简介）
    ↓
品牌色提取 + 主体推导 + 风格匹配
    ↓
Prompt 组装（风格规则 + 品牌层级 + 安全约束）
    ↓
生图 API → 压缩至 ≤200KB → 输出
```

风格匹配优先级：`--style N` 强制 > 分类推荐池 > 全局轮换兜底。完整的 12 种风格说明和分类映射表见 [docs/style-guide.md](docs/style-guide.md)。
