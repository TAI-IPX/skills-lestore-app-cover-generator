# App Store Cover Generator

输入 App 信息，自动生成应用商店封面展示图。

## 它能做什么

只需提供 App 名称、分类和一句话介绍，自动完成：

- **信息补全** — 信息不足时自主推断视觉方向，不追问
- **风格匹配** — 内置 18 种视觉风格，每次出图自动轮换确保多样性
- **主体推导** — 从核心功能提取差异化视觉符号
- **质量自检** — 自动评分，不达标自动修正
- **生图输出** — 调用配置的 API 直接出图

## 效果示例

> 输入：美团 · 生活服务 · 外卖美食买菜打车酒店火车票
>
> 输出：应用商店封面图

## 能处理多模糊的输入

| 输入信息 | 处理方式 |
|---------|---------|
| 名称 + 分类 + 简介 | 直接生成 |
| 名称 + 分类 | 语义推断补全 |
| 只有名称或只有分类 | 双引擎推断 |
| 一句模糊描述 | 生成 3 个风格方向供选择 |

## 配置

在项目目录创建 `.env`，填入你的生图 API：

```env
IMAGE_API=openai
IMAGE_BASE_URL=https://api.openai.com
IMAGE_API_KEY=sk-xxx
IMAGE_MODEL=dall-e-3
```

支持任何 OpenAI 兼容的 `/v1/images/generations` 端点。未配置时自动引导。
